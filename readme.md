# Epidemic Spread Simulation

A epidemic simulation project built with Python and Javascript.

This project simulates how infectious diseases spread through a population using the SIR/SIRD epidemiological model. Users can change parameters such as infection rate, recovery rate, death rate, and population size to observe how the outbreak evolves over time through interactive graphs.

## Features

- Simulate disease spread over time
- Adjustable parameters:
  - Susceptible population
  - Infected population
  - Beta (spread rate)
  - Gamma (recovery rate)
  - Mu (death rate)
  - Simulation days
- Interactive graphs using Chart.js
- Real-time visualization of:
  - Susceptible
  - Infected
  - Recovered
  - Deceased

## Technologies Used

- Python
- Flask
- JavaScript
- HTML/CSS
- Chart.js
